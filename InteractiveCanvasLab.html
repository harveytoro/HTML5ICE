Interactive canvas with Kinectjs

"KineticJS is an HTML5 Canvas JavaScript library that extends the 2d context by enabling canvas interactivity for desktop and mobile applications."
The example below is a simple noughts and crosses game you can drag the noughts or crosses to the position you wish to play double click and there position is then fixed (draggable property is set to false), the example does not tell you which player has won.
