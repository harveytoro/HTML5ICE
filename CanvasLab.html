<canvas>

The HTML5 spec defines The canvas element provides scripts with a resolution-dependent bitmap canvas, which can be used for rendering graphs, game graphics, or other visual images on the fly.
Below are two canvas examples the first on the left uses "lineTo(x,y)" and "moveTo(x,y)" to draw a line between two points and the second on the right uses "arc(x,y,radius,DrawAngleStart,DrawAngleEnd,Clockwise-TrueorFalse)" to draw a circle.

Play with the code

Try spelling out your name or drawing a picture. The above example is basic try experimenting and seeing what you come up with.

Useful to know

To draw a rectangle: "context.rect(x,y,width,height);"
To draw a rectangle with a fill 'color': "context.rect(x,y,width,height);"
To draw a line: "context.lineTo(x,y);"
To draw an arc along a circle "context.arc(x,y,radius,DrawAngleStart,DrawAngleEnd,Clockwise-TrueorFalse)"
To add text to the canvas "context.fillText("text",x,y);"
There are many other methods to the canvas element so check them out at the Mozilla Developer Network Site.
