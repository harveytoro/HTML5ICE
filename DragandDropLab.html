<h1 draggable="true">Drag and Drop</h1>

The HTML5 spec defines an event-based drag-and-drop mechanism. All HTML elements may have the draggable content attribute set and all HTML elements may have the dropzone content attribute set.
The Example is a set of patterns from subtile Patterns website. Drag a pattern from the left and drop the pattern on the dropzone target which in this case in the word "Background" on the right to change the background pattern of the body.
